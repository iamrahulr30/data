We are happy to announce the latest April 2025 patch & security update release for the Microsoft Build of OpenJDK. Download and install (https://aka.ms/msopenjdk-dl) the binaries today. OpenJDK 21.0.7 OpenJDK 17.0.15 OpenJDK 11.0.27 Check our release notes page (https://learn.microsoft.com/en-us/java/openjdk/release-notes) for details on fixes and enhancements. The source code of our builds is available now on GitHub for further inspection: jdk21u (https://github.com/microsoft/openjdk-jdk21u/tree/release/jdk-21.0.7_6) , jdk17u (https://github.com/microsoft/openjdk-jdk17u/tree/release/jdk-17.0.15_6) , jdk11u (https://github.com/microsoft/openjdk-jdk11u/tree/release/jdk-11.0.27_6) . Container images will also be updated, and soon we will be upgrading the base OS layer. Read this other announcement for details (https://devblogs.microsoft.com/java/important-updates-to-container-images-of-microsoft-build-of-openjdk/) . Microsoft Build of OpenJDK specific updates OpenJDK21 Backport of JDK-8350609 (https://bugs.openjdk.org/browse/JDK-8350609) : Cleanup unknown unwind opcode (0xB) for windows (Enhancement – P3) Enabled GPU selection hinting in Windows builds using NvOptimusEnablement and AmdPowerXpressRequestHighPerformance Exported driver hints to prefer high-performance GPU (NVIDIA/AMD) from the JVM Useful in environments like Minecraft, IDEs, or ML tooling to ensure GPU utilization OpenJDK17 Backport of JDK-8350609 (https://bugs.openjdk.org/browse/JDK-8350609) : Cleanup unknown unwind opcode (0xB) for windows (Enhancement – P3) OpenJDK11 No Microsoft-specific updates for this release. Special notes This release of the Microsoft Build of OpenJDK we have started shipping packages for SUSE Linux. Please see our Install page (https://learn.microsoft.com/en-us/java/openjdk/install#install-on-suse) for details. Summary of Upstream Updates OpenJDK 21 List of fixes (https://bugs.openjdk.org/issues/?jql=project%20%3D%20JDK%20AND%20status%20in%20(Closed%2C%20Resolved)%20AND%20fixVersion%20%3D%2021.0.7) See the OpenJDK Vulnerability Advisory (https://openjdk.org/groups/vulnerability/advisories/2025-04-15) OpenJDK 17 List of fixes (https://bugs.openjdk.org/issues/?jql=project%20%3D%20JDK%20AND%20status%20in%20(Closed%2C%20Resolved)%20AND%20fixVersion%20%3D%2017.0.15) See the OpenJDK Vulnerability Advisory (https://openjdk.org/groups/vulnerability/advisories/2025-04-15) OpenJDK 11 List of fixes (https://bugs.openjdk.org/issues/?jql=project%20%3D%20JDK%20AND%20status%20in%20(Closed%2C%20Resolved)%20AND%20fixVersion%20%3D%2011.0.27) See the OpenJDK Vulnerability Advisory (https://openjdk.org/groups/vulnerability/advisories/2025-04-15) OpenJDK 8 We continue to provide support on Azure and internally at Microsoft for OpenJDK 8 binaries of Eclipse Temurin built by the Eclipse Adoptium project. To facilitate its usage, we ship container images of OpenJDK 8 (https://mcr.microsoft.com/en-us/product/openjdk/jdk/tags) on top of CBL-Mariner and Ubuntu. Refer to our documentation.
